# Socket.IO Client Tester

A lightweight, browser-based tester for Socket.IO servers.  
Easily connect as any user, send custom messages to other users, and monitor incoming events in real time. Ideal for testing local, staging, or remote Socket.IO servers with a clean and responsive interface.

---

## Features

- Connect to any Socket.IO server via custom URL  
- Specify user ID for connection  
- Send messages to target users  
- Real-time event logging  
- Modern, responsive UI  
- Works with local, staging, or remote servers

---

## Installation

1. Clone the repository:

```bash
git clone https://github.com/orchid0610/socket-client.git
cd socket-client
```
2. Serve with php:
```bash
php -S 0.0.0.0:8000
```
